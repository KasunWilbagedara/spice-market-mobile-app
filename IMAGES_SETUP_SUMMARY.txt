## **How to Add Custom Logo**

### **Option 1: Use Built-in Logo** (Recommended - No Files Needed)
The app already has a professional programmable logo. Just use it!

### **Option 2: Add Your Custom Logo File**

**Step 1:** Create `assets/images/` folder
```
flutter_application/
â””â”€â”€ assets/
    â””â”€â”€ images/
```

**Step 2:** Add your logo file
```
Supported formats: PNG, JPG, SVG, WebP
```

**Step 3:** Update `pubspec.yaml`
```yaml
flutter:
  uses-material-design: true
  assets:
    - assets/images/
```

**Step 4:** Use in code
```dart
Image.asset('assets/images/your_logo.png', width: 100)
```

**Step 5:** Run commands
```bash
flutter clean
flutter pub get
flutter run
```

## **How to Add Product Photos**

### **For Sellers (Uploading)**
1. Go to **Seller Home**
2. Click **Add Spice**
3. Tap the **Image Upload Area**
4. Choose **Camera** or **Gallery**
5. Select/Take photo
6. Photo appears in preview
7. Submit to add product

### **For Buyers (Viewing)**
Products with images show in the grid. Without images, shows spice icon placeholder.

## **Permissions Required**

### **Android** (`android/app/src/main/AndroidManifest.xml`)
```xml
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
```

### **iOS** (`ios/Runner/Info.plist`)
```xml
<key>NSCameraUsageDescription</key>
<string>App needs camera access to take photos</string>
<key>NSPhotoLibraryUsageDescription</key>
<string>App needs photo library access</string>
```
